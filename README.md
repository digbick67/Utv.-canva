# Canva AI Analysis

## Beskrivning
Detta projekt utvärderar Canva AI baserat på tre typer av bildgenerering:
- Kreativ bild
- Infografik
- Planlösning

## Slutsats
AI:n är stark på visuella och kreativa uppgifter, men svag på:
- Text
- Data
- Struktur

## Hur man kör
1. Klona repot
2. Öppna `index.html` i webbläsaren
